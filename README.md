<a name="features">🔋 Features</a>
👉 Authentication System: A robust authentication system ensuring user security and privacy

👉 Explore Page: A homepage for users to explore posts, with a highlighted section for top creators

👉 Like and Save Features: Allows users to like and save posts, with dedicated pages to manage liked and saved content

👉 Detailed Post Page: A detailed post page displaying content and related posts for an immersive user experience

👉 Profile Page: A user profile page showing liked posts and offering options to edit the profile

👉 Browse Other Users: Allows users to browse and explore other users' profiles and posts

👉 Post Creation Page: Implement a user-friendly post creation page with easy file management, storage, and drag-and-drop feature

👉 Post Editing Functionality: Provides users with the ability to edit their post content at any time

👉 Responsive User Interface with Bottom Bar: A responsive user interface with a bottom bar, enhancing the mobile app feel for seamless navigation

👉 React Query Integration: Incorporates the React Query (Tanstack Query) data fetching library for, Automatic caching to improve performance, Parallel queries for efficient data fetching, First-class mutations, etc.

👉 Backend as a Service (BaaS) - Appwrite: Utilize Appwrite as a Backend as a Service solution for simplified backend development, offering features like authentication, database, file storage, and more

and much more, including code architecture and reuse.
